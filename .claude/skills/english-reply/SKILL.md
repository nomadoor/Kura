# English Reply

Use this skill for English communication.

## Style

- Be concise and technically precise.
- Separate observed facts from guesses.
- Include commands, versions, and run IDs only when useful.
- Do not expose local private paths, tokens, paid account details, or unpublished dataset contents.
- If asking upstream, provide minimal reproduction steps and exact error messages.
